## 📌 簡介

以星空為主題打造的靜態網頁，包含自我介紹、作品集展示、影片精選、新詩專欄與聯絡資訊。

## 🛠️ 技術棧

- **HTML5 / CSS3 / JavaScript (Vanilla)**
- Google Fonts（Noto Sans TC / Zen Maru Gothic）
- Canvas 星空粒子動畫
- JSON 資料驅動（動態載入內容）
- GitHub Pages 部署

## 📂 專案結構

```
├── index.html          # 主頁面
├── css/style.css       # 樣式表
├── js/main.js          # 主程式邏輯
├── data/
│   ├── about.json      # 個人介紹資料
│   ├── projects.json   # 作品集 & 影片資料
│   └── poems.json      # 新詩資料
├── images/             # 圖片素材
├── audio/              # 背景音樂
└── video/              # 影片素材
```

## ✨ 功能特色

- 🌌 星空粒子背景動畫與流星效果
- 🎵 背景音樂播放控制
- 📱 響應式設計，支援各種裝置
- 📦 資料與畫面分離，以 JSON 管理內容
